# Romeo-Juliet-Tracker
This script has all the essential knowledge of the characters written in the play 'Romeo and Juliet' by Shakespear ... all you have to do is write the name and it will tell its essential background inside the console

#Problem:
The problem i was facing was keeping the track of each character while reading the play so i wrote this simple script and hard coded all the essential knowledge of each character.

#Accomplishment:
This is my first project where I:
->Solved a personal problem with code
->Documented specifications
->Created a step-by-step plan and executed tasks

#How to use:
1.Run the script in the console.
2.Type the name of a character to get their background.
3.Type list to see all character names with correct spelling.
4.Type x to exit the program.
